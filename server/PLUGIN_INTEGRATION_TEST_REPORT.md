## 已知限制与注意事项

1. **LLM 延迟：** 记忆创建时间主要由 LLM 事实提取主导 (每次调用 3-5 秒)。这是预期行为，可通过以下方式优化：
   - 使用更快的 LLM 模型
   - 实现本地 LLM 部署
   - 尽可能批量处理操作

2. **向量搜索：** 向量相似度搜索返回语义相关的结果，而非精确匹配。这是语义记忆检索的预期行为。

3. **图记忆：** 临时禁用以支持 agent_id-only 查询，确保与 Plugin 使用模式的兼容性。

4. **速率限制：** 当前配置为每个 API Key 每 60 秒 200 个请求。根据生产负载要求进行调整。
