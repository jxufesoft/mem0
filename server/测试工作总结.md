**修复文件：** `server/main.py` (第 565-587 行)

**问题：** 记忆更新端点在接收更新请求时报错 `'dict' object has no attribute 'replace'`

**根本原因：** 端点直接将请求字典传递给 `Memory.update()` 方法，但该方法期望字符串参数

**解决方案：** 修改端点代码以从请求字典中正确提取字符串数据，支持多种请求格式
